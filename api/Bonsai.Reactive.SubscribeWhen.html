Class SubscribeWhen

Namespace
Bonsai.Reactive
Assembly
Bonsai.Core.dll

Represents an operator that subscribes to the first sequence only after the second sequence emits a notification.

The SubscribeWhen operator modifies the source sequence by waiting for the first value emitted by the second sequence before subscribing to the first sequence. This has the effect of delaying the start of the sequence until the second sequence emits a value.

SubscribeWhen can be used to control the timing of initialization. Note that if a source is hot, delaying the start of the sequence will not preserve past data.
